# Simulation Without Tears 
Java/Maven project experimenting with Alloy-based simulation.

This is an early-stage project and the design, features, and structure are likely to change.

**For now:**
- transforms an OntoUML-model (json) to alloy code using ontouml-js.ontouml2alloy
- lets the user choose/configure a simulation constraint 
- launches alloy visualizer with an optional theme
